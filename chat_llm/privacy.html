개인정보 처리방침

AI 톡비서는 사용자의 개인정보를 보호하기 위해 최선을 다합니다. 본 방침은 2024년 1월 1일부터 적용됩니다.

🔒 보안 강화 조치

  • 서버에 원문을 저장하지 않음: 모든 대화 내용은 서버에 저장되지 않으며, 요약 요청 시에만 임시로 처리됩니다.
  • 비식별화 마스킹 처리: 전화번호는 [PHONE], 이메일은 [EMAIL], 계좌번호는 [ACCOUNT], 주민번호는 [SSN], 주소는 [ADDRESS] 토큰으로 자동 치환됩니다.
  • 모든 데이터는 기기 로컬에만 보관: 수집된 대화 내용은 사용자의 기기 내부 저장소에만 보관되며, 외부로 전송되지 않습니다.

1. 개인정보 수집 및 이용

1.1 수집하는 개인정보

  • 알림 접근 권한: 메신저 알림을 읽어 대화 내용을 수집합니다.
  • 수집된 정보: 대화방 이름, 발신자, 메시지 내용, 시간 정보
  • 이용 목적: AI 기반 대화 요약 서비스 제공

2. 개인정보 보호 조치

2.1 자동 마스킹 처리

  • 모든 개인정보는 LLM 서버로 전송 전에 자동으로 비식별화 마스킹 처리됩니다.
  • 전화번호는 [PHONE], 이메일은 [EMAIL], 계좌번호는 [ACCOUNT], 주민번호는 [SSN], 주소는 [ADDRESS] 토큰으로 자동 치환됩니다.

2.2 암호화 통신

  • 모든 통신은 HTTPS로 암호화되어 전송됩니다.
  • SSL/TLS 인증서를 통한 안전한 통신을 보장합니다.

2.3 서버 데이터 보관

  • 서버에 원문을 저장하지 않습니다. 요약 요청 시에만 마스킹된 정보가 임시로 전송되며, 요약 완료 후 즉시 삭제됩니다.

3. 데이터 저장

3.1 로컬 저장

  • 수집된 대화 내용은 앱 내부 저장소에만 저장되며, 외부로 전송되지 않습니다.
  • 모든 데이터는 기기 로컬에만 보관됩니다.
  • LLM 요약 요청 시에만 마스킹된 정보가 서버로 전송됩니다.

4. 권한 사용

  • 배터리 최적화 제외: 백그라운드에서 알림을 지속적으로 수신하기 위해 필요합니다.
  • 알림 접근 권한: 메신저 알림을 읽어 대화 내용을 수집하기 위해 필요합니다.

5. 사용자 권리

  • 언제든지 앱 설정에서 권한을 해제할 수 있습니다.
  • 수집된 데이터는 앱 삭제 시 함께 삭제됩니다.
  • 개인정보 처리에 대한 문의는 앱 내 고객지원을 통해 가능합니다.

6. 개인정보 처리방침 변경

본 방침은 관련 법령 및 정책 변경에 따라 수정될 수 있으며, 변경 시 앱 내 공지사항을 통해 안내드립니다.

본 방침은 2024년 1월 1일부터 시행됩니다.
